# Projeto Livraria MVC com PHP

Este é um projeto de uma livraria desenvolvido em PHP com o padrão MVC, utilizado como atividade para a faculdade.

## 📚 Funcionalidades

- Cadastro de livros
- Lista de clientes
- Tela de vendas
- Dashboard com gráficos

## 🛠 Tecnologias utilizadas

- PHP
- MySQL
- HTML / CSS
- Bootstrap

## 🚀 Como rodar o projeto

- Clone o repositório
- Configure o banco de dados com o script fornecido
- Execute com um servidor local como XAMPP ou WAMP
